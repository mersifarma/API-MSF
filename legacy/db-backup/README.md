## Skema yang perlu disesuaikan saat migrasi ke PostgreSQL

Source MariaDB → target PostgreSQL. Yang perlu diubah saat porting:

| MariaDB | PostgreSQL |
|---------|------------|
| `int(11) AUTO_INCREMENT` | `SERIAL` / `INTEGER GENERATED BY DEFAULT AS IDENTITY` |
| `tinyint(1)` | `BOOLEAN` |
| `datetime` | `TIMESTAMP` (atau `TIMESTAMPTZ` kalau perlu TZ) |
| `ENGINE=InnoDB ... CHARSET=utf8` | hilangkan (Postgres default UTF-8) |
| Backtick `` ` `` untuk identifier | double quote `"` (atau hilangkan kalau lowercase) |
| `UNSIGNED` | hilangkan (pakai CHECK constraint kalau perlu) |
| `IFNULL(x, y)` | `COALESCE(x, y)` |
| `FIND_IN_SET(x, csv_col)` | `x = ANY(string_to_array(csv_col, ','))` — atau lebih baik normalisasi ke array/junction table |
| `GROUP_CONCAT(...)` | `string_agg(..., ',')` |
| `DATE_FORMAT(x, '%Y-%m')` | `to_char(x, 'YYYY-MM')` |
| `NOW() - INTERVAL 30 MINUTE` | `NOW() - INTERVAL '30 minutes'` |
| `\\` escape di string | `E'...'` atau pakai dollar quoting |
| `LOCK TABLES ... WRITE; ... UNLOCK TABLES;` | hilangkan (Postgres pakai BEGIN/COMMIT) |

Tool yang bisa bantu konversi otomatis: **`pgloader`** (https://pgloader.io/) — mendukung MySQL/MariaDB → PostgreSQL langsung. Atau **`mysql-to-postgresql-converter`** untuk script-based.

## Cara restore (kalau perlu kembali ke MariaDB local)

```bash
mysql -u root -p monitoring < users.sql
mysql -u root -p monitoring < data_pegawai.sql
# urutan ideal: master dulu (users, data_pegawai, app_modul, struktur, list_dokter_visit_new, data_product, data_spec_dr)
# lalu transactional (call_list, call_plan_actual, dll.)
```

Atau gabung semua:

```bash
cat *.sql > all.sql
mysql -u root -p monitoring < all.sql
```

> `_extract.awk` adalah script yang dipakai untuk ekstraksi (tidak perlu di-restore).

## Privacy / sensitivity

File ini berisi **data produksi real** (nama pegawai, dokter, alamat, password hash). Jangan commit ke repo public — sudah di-`.gitignore`-kan? Cek:

```bash
git check-ignore db-backup/
```

Kalau belum, tambahkan ke `.gitignore`:

```
/db-backup/*.sql
```
